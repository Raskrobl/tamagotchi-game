# tamagotchi-game
Моя игра Тамагочи
## Защита Windows SmartScreen

Если Windows блокирует запуск:
1. Нажмите "Подробнее"
2. Выберите "Выполнить в любом случае"
3. Или добавьте исключение в Защитнике

4.Игра на телефоне и пк (на пк если боитесь скачивать .exe) https://Raskrobl.github.io/tamagotchi-game

5.Светлая и тёмная настройка игры не работает(кроме версии для пк .exe) (добавлю в следующем глобальном  обновлении)
